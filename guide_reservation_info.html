N/1 결제로 불편함을 없앴습니다.

1

예약 가능 잔여팀 확인 후 원하시는 날짜와 시간을 선택해주세요.

로그인(회원가입) 필수입니다.

루나엑스에서는 다양한 예약이 가능합니다.
-6홀 / 12홀 / 18홀 / 24홀

2

예약을 진행하기 위해필수 사항들을 체크해주세요.

모두 동의해야 예약이 가능합니다.

3

선택한 인원만큼 결제 요청 링크를 전달하기 위해 결제를 진행하는 분들의
이름과 전화번호를입력해주세요.

1인 전액 결제는 해당 단계가 생략됩니다.

버튼을 누르면 최근 함께 플레이한
동반자 정보를 간편하게 불러올 수도 있습니다.

입력 후 '결제 요청 문자 보내기' 버튼을 누르면
해당 번호로 결제 링크가 발송됩니다.
(문자메시지 혹은 카카오 알림톡)

4

결제 기한까지 결제자 모두 신용카드 정보를 등록하고 예약 신청을 해두면
결제 기한에 일괄 승인되며 예약이 확정됩니다.

즉시 결제되지 않으며 ‘결제 기한’에 모든 결제 정보가 입력되면 결제가 진행됩니다. 결제 기한까지 누락된 결제 요청이 있을 경우 예약은 자동 취소 됩니다.

만약, 결제 기한까지 미 결제자가 있거나 승인거절 신용카드를 사용한 경우 예약이 취소될 수 있으니 확인 바랍니다.

결제 기한 전까지는 취소가 가능하지만 결제 기한 이후, 결제 승인 후에는 취소가 어려울 수 있으니 신중하게 예약 부탁드립니다.
